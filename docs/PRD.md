## 1. 產品概述

### 1.1 產品簡介

氣體/液體幫浦測試平台是一套自動化測試系統，用於在幫浦上產線安裝前進行規格驗證與品質檢測。系統可測試真空幫浦、正壓幫浦及液體幫浦，並提供完整的電力消耗、壓力、流量等數據分析。

### 1.2 產品定位

- **用途**: 生產線品質管制測試設備
- **應用場景**: 半導體製造設備維護、幫浦品質驗證
- **目標市場**: 半導體製造業、精密機械製造業

### 1.3 核心價值

1. **上線前品質保證**: 確保幫浦符合規格才上產線
2. **數據化管理**: 建立幫浦測試數據庫供比對分析
3. **功率監測**: 新增電力消耗測定功能，評估能源效率
4. **自動化測試**: 減少人工操作，提高測試效率與準確性

## 2. 目標與目的

### 2.1 業務目標

- 降低因幫浦故障導致的產線停機時間
- 建立標準化幫浦測試流程
- 提升幫浦品質管理水準
- 累積幫浦性能數據庫供後續分析

### 2.2 產品目標

- 支援 50+ 種不同型號幫浦測試
- 測試準確度達 ±2% 以內
- 單次測試時間控制在 1 小時內
- 資料儲存格式標準化 (CSV)

### 2.3 使用者目標

- 操作介面簡單直覺，5 分鐘內完成參數設定
- 測試過程自動化，減少人工介入
- 測試報告清晰易懂，可快速判斷合格與否

#### FR-004: 自動測試模式

**優先級**: P0 (必須)

**需求描述**:

**真空幫浦測試模式**:
1. 自動控制電磁閥 A/C 關閉, B/D 開啟
2. 監測負壓壓力變化
3. 判斷壓力恆定條件:
   - 壓力變化 < 1% 持續 5 分鐘 → 停止測試
   - 壓力持續變化超過 1 小時 → 超時停止
4. 自動洩壓並記錄數據

**正壓幫浦測試模式**:
1. 自動控制電磁閥 B/D 關閉, A/C 開啟
2. 監測正壓壓力變化
3. 判斷壓力恆定條件 (同上)
4. 自動洩壓並記錄數據

**驗收標準**:
- [ ] 電磁閥切換正確無誤
- [ ] 壓力恆定判斷邏輯正確
- [ ] 超時機制運作正常
- [ ] 測試完成自動洩壓

#### FR-006: 安全保護機制

**優先級**: P0 (必須)

**需求描述**:

**急停開關**:
- 系統通電即開始監控
- 觸發時立即:
  - 停止馬達供電
  - 儲氣筒洩壓
  - 鎖定所有操作
  - 螢幕顯示 "緊急停止"
- 解除急停後方可繼續操作

**測試蓋開關**:
- 測試蓋未關閉無法啟動測試
- 測試中開蓋:
  - 停止馬達供電
  - 進入暫停狀態
  - 系統仍可操作
  - 蓋上後可繼續

**過載保護**:
- 監測電流超過設定值時:
  - 停止測試
  - 馬達停止
  - 儲氣筒洩壓
  - 顯示 "超載停止"

**電源不符檢測**:
- 偵測實際供電與設定不符時停止測試
- 顯示 "Pump 電力參數與配接電力不符"

**壓力上限保護**:
- 正壓達 8 kg/cm² 時暫停馬達
- 壓力降低後自動恢復
- 負壓達 -100 kPa 時暫停馬達

**驗收標準**:
- [ ] 急停功能在 100ms 內響應
- [ ] 所有安全機制經測試確認有效
- [ ] 警示訊息正確顯示

#### FR-008: 測試參數設定

**優先級**: P0 (必須)

**需求描述**:

**測試前設定流程**:
1. 選擇測試模式 (真空/正壓/手動)

**可調整參數**:
- 壓力恆定判斷時間 (預設 5 分鐘)
- 測試超時時間 (預設 1 小時)
- 正壓上限 (預設 8 kg/cm², 最高 8 kg/cm²)
- 負壓上限 (預設 -100 kPa)

**驗收標準**:
- [ ] 參數設定流程順暢

**優先級**: P0 (必須)

**需求描述**:

**硬體組件**:

- 0.5L 氣體儲氣筒 x2 (A, B)

- 負壓壓力計 (0 - -100 kPa)

- 逆止閥
- 洩壓消音器
- 末端空氣過濾器 x2

**管路配置**:
- 輸入端: 儲氣筒 A → 電磁閥 A → 過濾器 → 幫浦輸入
- 輸出端分支:
  - 分支 1: 幫浦輸出 → 三通 → 電磁閥 B → 流量計 → 過濾器
  - 分支 2: 三通 → 電磁閥 C → 逆止閥 → 儲氣筒 B → 壓力計
  - 分支 3: 三通 → 電磁閥 D → 洩壓消音器

**驗收標準**:
- [ ] 管路耐壓測試通過 (≥10 kg/cm²)
- [ ] 無洩漏 (壓降 < 1% / 10分鐘)
- [ ] 電磁閥切換時間 < 1 秒

## 5. 非功能需求

### 5.1 性能需求

| 需求項目 | 指標 |
|---------|------|
| **系統響應時間** | 使用者操作響應 < 100ms |
| **數據採樣頻率** | ≥ 1 Hz |
| **曲線更新頻率** | ≥ 10 FPS |
| **通訊延遲** | RS485 指令延遲 < 100ms |
| **測試準確度** | 壓力 ±1%, 電流 ±2%, 流量 ±2% |
| **連續運轉時間** | ≥ 24 小時無故障 |

### 5.2 可靠性需求

- **系統可用性**: ≥ 99%
- **MTBF (平均故障間隔)**: ≥ 2000 小時
- **MTTR (平均修復時間)**: ≤ 2 小時
- **數據備份**: 自動每日備份

### 5.3 可維護性需求

- **模組化設計**: 各子系統獨立可更換
- **故障診斷**: 提供系統自檢功能
- **錯誤日誌**: 記錄所有系統錯誤與警告
- **校正功能**: 提供感測器校正介面

### 5.4 可用性需求

- **學習曲線**: 新手 30 分鐘內學會基本操作
- **操作步驟**: 標準測試流程 ≤ 10 步
- **錯誤預防**: 提供操作提示與確認機制
- **多語言支援**: 繁體中文 (主要), 英文 (選用)

### 5.5 安全性需求

- **實體安全**:
  - 急停開關符合 ISO 13850 標準
  - 測試蓋具備安全聯鎖
  - 高壓管路防護罩

- **電氣安全**:
  - 過載保護
  - 短路保護
  - 接地保護
  - 符合 IEC 61010 標準

- **操作安全**:
  - 危險操作需二次確認
  - 測試中禁止拆卸
  - 洩壓後方可開蓋

### 5.6 環境需求

| 環境參數 | 規格 |
|---------|------|
| **操作溫度** | 10°C ~ 40°C |
| **儲存溫度** | -10°C ~ 60°C |
| **相對濕度** | 20% ~ 80% (非凝結) |
| **供電電源** | AC 110V/220V ±10%, 50/60 Hz |

#### 6.1.2 感測器規格

- 正壓型: 0 ~ 10 kg/cm², 精度 ±1% FS
- 負壓型: 0 ~ -100 kPa, 精度 ±1% FS
- 輸出: 4-20mA 或 RS485

**流量感測器**:

## 7. 系統操作流程

### 7.1 標準測試流程

```
[1] 開啟系統電源
  └─> 檢查系統自檢是否通過

[2] 啟動 PC 測試軟體
  └─> 確認 RS485 通訊正常

[3] 進入參數設定畫面
  ├─> 選擇測試模式 (真空/正壓/手動)
  ├─> 輸入幫浦型號
  ├─> 選擇供電電源 (AC/DC 及電壓)
  ├─> 輸入額定電力消耗
  ├─> (選用) 修改測試參數
  └─> 儲存設定

[4] 安裝待測幫浦
  ├─> 將幫浦固定於測試平台
  ├─> 連接進氣/出氣管路
  ├─> 連接電源線至指定接頭
  └─> 確認管路連接緊密

[5] 關閉測試蓋
  └─> 確認安全聯鎖指示燈亮起

[6] 執行測試前檢查
  ├─> 確認管路無洩漏
  ├─> 確認電源選擇正確
  └─> 確認急停開關正常

[7] 按下 [啟動測試] 按鈕
  └─> 系統自動執行測試流程

[8] 監控測試過程
  ├─> 觀察壓力曲線變化
  ├─> 觀察電流消耗數值
  └─> 留意警示訊息

[9] 測試完成
  ├─> 系統自動洩壓
  ├─> 顯示測試結果
  └─> 確認是否儲存至資料庫

[10] 拆卸幫浦
  ├─> 確認壓力歸零
  ├─> 打開測試蓋
  ├─> 拆除電源線
  ├─> 拆除管路
  └─> 取出幫浦

[11] 測試完成
```

### 7.2 緊急停止流程

```
[緊急情況發生]
  └─> 按下急停開關

[系統立即響應]
  ├─> 馬達停止供電
  ├─> 儲氣筒洩壓
  ├─> 鎖定所有操作
  └─> 螢幕顯示 "緊急停止"

[排除緊急情況]

[旋轉急停開關解除]
  └─> 系統恢復可操作狀態

[檢查系統狀態]
  └─> 確認無異常後可繼續操作
```

## 11. 假設與限制

### 11.1 假設條件

1. 客戶場地提供穩定的 AC 110V/220V 單相及三相電源
2. 測試環境溫濕度符合設備操作範圍
3. 待測幫浦具備標準管路接頭 (可轉接)
4. 操作人員具備基本電腦操作能力
5. 網路連線可用 (非必要，僅供 FTP 傳輸)

### 11.2 系統限制

1. **壓力範圍**:
   - 正壓最高 8 kg/cm²
   - 負壓最低 -100 kPa
   - 超出範圍的幫浦無法測試

2. **電源規格**:
   - AC: 110V/220V 單相, 220V 三相
   - DC: 12V/24V (可擴充至 48V)
   - 超出範圍需客製

3. **幫浦尺寸**:
   - 測試平台尺寸有限，超大型幫浦可能無法安裝

4. **測試時間**:
   - 手動模式最長 12 小時
   - 自動模式最長 1 小時 (超時保護)

5. **同時測試**:
   - 一次僅能測試一台幫浦
